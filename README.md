# Financial Statement Analysis & Excel Dashboard

**Data source:** Apple Inc. Annual Reports (Form 10-K) — FY2024 & FY2025  
**Units:** USD (millions)

## Overview
This project analyzes Apple Inc.’s financial statements to evaluate profitability, liquidity, leverage, and cash flow sustainability.  
An Excel dashboard visualizes year-over-year trends.

## Key Insights

- Profitability remained relatively stable year-over-year, with consistent operating margins.
- Liquidity remained tight, indicating careful working capital management.
- Leverage increased, suggesting higher reliance on debt financing.
- Free cash flow improved, demonstrating enhanced cash generation capacity.
